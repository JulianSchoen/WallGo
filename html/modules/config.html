Config

class WallGo.Config(configGrid: ~WallGo.config.ConfigGrid = <factory>, configEOM: ~WallGo.config.ConfigEOM = <factory>, configHydrodynamics: ~WallGo.config.ConfigHydrodynamics = <factory>, configThermodynamics: ~WallGo.config.ConfigThermodynamics = <factory>, configBoltzmannSolver: ~WallGo.config.ConfigBoltzmannSolver = <factory>)[source]

Data class that holds all the model-independent configs. It contains objects of the data classes ConfigGrid, ConfigEOM, ConfigEffectivePotential, ConfigHydrodynamics, ConfigThermodynamics and ConfigBoltzmannSolver. It can also load the configs from an .ini file.
